# PARTİ 1 — TEMELLER (Böl. 1-8)

## 1. CÜMLE YAPISI (SVO)

- Endonezce **özne-yüklem-nesne (SVO)** dizilişlidir: `Saya makan nasi` (Ben
  pirinç yerim).
- Fiiller **çekimlenmez** — zaman/kişi için ek almaz; zaman, **belirteçlerle**
  (sudah, sedang, akan…) belirtilir.
- Cinsiyet yok, artikel yok.

## 2. İSİM ÖBEKLERİ (baş-başta)

Endonezcede niteleyen, nitelenenin **ARDINDAN** gelir (Türkçenin tersi):

- **İsim + iyelik**: rumah **saya** (benim evim), buku **dia** / buku**nya**.
- **İsim + isim** (niteleme): guru **bahasa** (dil öğretmeni), tukang **kayu**
  (marangoz).
- **İsim + sıfat**: rumah **besar** (büyük ev), orang **baik** (iyi insan).
  Sıfat ASLA isimden önce gelmez.
- **İsim + işaret**: buku **ini** (bu kitap), orang **itu** (o kişi).
- **İsim + fiil/ilgi tümcesi**: orang **yang datang** (gelen kişi).

## 3. "OLMAK": adalah / ialah

- İki ismi eşitlerken **adalah** (yazılı/resmî) ya da **ialah**; çoğu zaman
  düşer: `Dia (adalah) guru` (O öğretmendir).
- **Sıfat veya fiilden önce adalah KULLANILMAZ**: `Dia pintar` (✓), `Dia adalah
  pintar` (✗). `Saya makan` (✓), `Saya adalah makan` (✗).
- **merupakan** = "teşkil eder / -dir": `Jakarta merupakan ibu kota` (Jakarta
  başkenttir).

## 4. ada — var / bulunmak / sahip olmak

- "Var": `Ada buku di meja` (Masada kitap var).
- "Bir yerde olmak": `Dia ada di rumah` (O evde).
- "Sahip olmak" (= punya/mempunyai): `Saya ada uang` / `Saya punya uang`.

## 5. SAYILAR, SAYAÇLAR, ÇOKLUK

- Sayılar: satu, dua, tiga, empat, lima, enam, tujuh, delapan, sembilan,
  sepuluh; puluh (10'lar), ratus (100), ribu (1.000), juta (milyon).
- **Sayı + sayaç + isim**: orang (insan), ekor (hayvan), buah (nesne), batang
  (uzun), helai (ince/yaprak), biji (küçük tanecik):
  - dua **orang** guru, tiga **ekor** kucing, lima **buah** buku.
- Çokluk: bağlamla ya da **ikileme** (buku-buku = kitaplar). **Sayıdan/
  niceleyiciden sonra ikileme YAPILMAZ**: tiga buku (✓), tiga buku-buku (✗).

## 6. OLUMSUZLUK

- **tidak** — fiil/sıfatı olumsuzlar: `Saya tidak makan`, `Dia tidak tinggi`.
- **bukan** — isim/zamiri olumsuzlar: `Ini bukan buku saya`, `Dia bukan guru`.
- **belum** — "henüz değil": `Saya belum makan`.
- **jangan** — "yapma" (olumsuz emir): `Jangan pergi`.
- Sözcük türüne göre tidak ↔ bukan doğru seçilmeli.

## 7. SORULAR

- Soru kelimeleri: **apa** (ne), **siapa** (kim), **(di) mana** (nerede),
  **kapan** (ne zaman), **bagaimana** (nasıl), **mengapa/kenapa** (neden),
  **berapa** (ne kadar/kaç).
- Evet/hayır sorusu: **apakah** ile ya da yükselen tonlama / **-kah** ekiyle:
  `Apakah kamu lapar?` / `Kamu lapar?`

## 8. ZAMİRLER VE EDATLAR

- Zamirler: saya/aku (ben), kamu/Anda/kau (sen/siz), dia/ia (o), kami (biz —
  dinleyen hariç), kita (biz — dinleyen dahil), mereka (onlar).
- İyelik ekleri: **-ku, -mu, -nya** (buku saya = buku**ku**; rumah dia =
  rumah**nya**).
- Edatlar: **di** (-de/-da, durağan), **ke** (-e/-a), **dari** (-den/-dan),
  **pada/kepada** (kişiye/zamana), **untuk** (için), **dengan** (ile),
  **oleh** (edilgen özne), **tentang** (hakkında).
- **Dikkat**: edat **di** ayrı yazılır (`di rumah`), edilgen önek **di-**
  fiile bitişir (`dibaca`).

# PARTİ 2 — FİİL MORFOLOJİSİ (Böl. 9-11)

## 9. GEÇİŞLİ VE GEÇİŞSİZ FİİLLER

- Çıplak kök çoğunlukla emir/konuşma dilidir; çoğu fiil **önek alır**.
- Geçişsiz/durum fiilleri genelde **ber-** veya **meN-**; etken geçişli fiiller
  **meN-** (etken) / **di-** (edilgen) alır.

## 10. ber- ÖNEKİ

- "Sahip olmak": ber + istri → **beristri** (evli/karısı olan).
- "Giymek/kullanmak": ber + baju → **berbaju** (giyinik).
- Kökün eylemini yapmak: **bekerja** (çalışmak), **berlari** (koşmak),
  **belajar** (öğrenmek).
- Allomorf: **bel-** (ajar → belajar), **be-** r ile başlayan kökte (kerja →
  bekerja).
- ber- + sayı: berdua (ikisi birlikte). ber-…-an: karşılıklı/dağınık eylem
  (bersalaman = el sıkışmak).

## 11. meN- ÖNEKİ VE SES DEĞİŞİMLERİ

Kökün ilk sesine göre nazalleşir; sessiz ünsüz (p, t, k, s) **düşer**:

| Önek | Kökün ilk sesi | Örnek |
|---|---|---|
| me- | l, m, n, r, w, y, ng, ny | lihat → **melihat** |
| mem- | b, f, **p**(düşer) | pukul → **memukul** |
| men- | d, c, j, **t**(düşer) | tulis → **menulis** |
| meng- | g, h, ünlüler, **k**(düşer) | kirim → **mengirim**, ambil → **mengambil** |
| meny- | **s**(düşer) | sapu → **menyapu** |
| menge- | tek heceli kök | cat → **mengecat** |

- meN- etken geçişli fiil yapar: `membaca buku` (kitap okumak); nesne fiilden
  sonra gelir. Bazı meN- fiilleri geçişsiz eylem fiilidir.

## 11b. di- EDİLGEN ÖNEKİ

- **di-** edilgen / nesne-odağı yapar: `Buku itu dibaca` (Kitap okundu).
  Özne **oleh** ile: `dibaca oleh saya` (3. kişi için tipik).
- di- fiile bitişir; edat di ile karıştırma.

# PARTİ 3 — EK PAKETİ (Böl. 12-18)

## 12. meN-…-kan

- Fiili geçişli yapar; iki ana işlev:
  - **Ettirgen** ("-leştirmek / -e neden olmak"): panas → **memanaskan**
    (ısıtmak), besar → membesarkan.
  - **Yararına** ("biri için yapmak"): beli → **membelikan** (biri için satın
    almak): `Saya membelikan adik buku` (Kardeşime kitap aldım).

## 13. meN-…-i

- Geçişli; nesnesi genelde **yer/hedef**; konumsal ya da yinelemeli:
  - tanam → **menanami** (üstüne ekmek): `menanami kebun dengan jagung`.
  - pukul → **memukuli** (defalarca vurmak).
- **Karşıtlık**: `menanam jagung` (mısır ekmek) ↔ `menanami kebun` (bahçeyi
  ekmek) — -kan nesnesi taşınan şey, -i nesnesi yerdir.

## 14. ter- ÖNEKİ

- **İstem dışı/kazara**: jatuh → **terjatuh** (kazara düşmek).
- **Sonuç-durumu**: tutup → **tertutup** (kapalı).
- **Yeterlilik** ve sıfatta **en üstünlük**: besar → **terbesar** (en büyük).

## 15. ke-…-an

- **Soyut isim**: bebas → **kebebasan** (özgürlük), indah → keindahan (güzellik).
- **Olumsuz/aşırı yaşantı fiili**: hujan → **kehujanan** (yağmura yakalanmak),
  panas → kepanasan (aşırı sıcaktan etkilenmek).

## 16. peN-, -an, peN-…-an / per-…-an

- **peN-** = fail/araç ismi (meN- ile aynı nazalleşme): tulis → **penulis**
  (yazar), bersih → **pembersih** (temizleyici).
- **-an** = sonuç/nesne ismi: makan → **makanan** (yemek), tulis → **tulisan**
  (yazı).
- **peN-…-an / per-…-an** = süreç/soyut isim: didik → **pendidikan** (eğitim),
  jalan → **perjalanan** (yolculuk), satu → persatuan (birlik).

# PARTİ 4 — ÇATI, ODAK VE -nya (Böl. 19-20, 7)

## 17. ÖZNE-ODAĞI VE NESNE-ODAĞI

- **Özne-odağı** (etken, meN-): `Saya membaca buku itu` (Kitabı okuyorum).
- **Nesne-odağı** iki kalıp:
  1. **3. kişi özne** → **di-** fiil (+ oleh): `Buku itu dibaca (olehnya)`.
  2. **1./2. kişi özne** → Nesne + özne-zamiri + **çıplak kök** (di- YOK):
     `Buku itu **saya baca**` (Kitabı ben okudum). saya/kamu gibi zamir
     öznesiyle ASLA di- kullanılmaz; zamir çıplak fiilin hemen önündedir.

## 18. yang

- İlgi belirteci "ki/-en": `orang **yang** datang` (gelen kişi),
  `buku **yang** saya beli` (aldığım kitap).
- İsimleştirir: `**yang** merah` (kırmızı olan).
- "Hangisi?": `**yang** mana?`

## 19. -nya KULLANIMLARI

- 3. kişi **iyelik**: rumah**nya** (onun evi).
- **Belirlilik "-(y)i/the"**: buku**nya** (o kitap, bilinen kitap).
- **İsimleştirme**: datang**nya** (gelişi).
- Kibarlık/konu işaretleme.

## 20. KARŞILAŞTIRMA (Böl. 7)

- **lebih … daripada** (-den daha …): `Dia lebih tinggi daripada saya`.
- **kurang … daripada** (-den daha az …).
- **paling …** veya **ter-** (en …): paling besar = terbesar.
- **sama … dengan** / **se-** (… kadar): `setinggi saya` (benim kadar uzun).

## 21. GÖRÜNÜŞ/ZAMAN BELİRTEÇLERİ (çekim yok)

Fiil çekilmez; zaman/görünüş **belirteçle** verilir, fiilden önce:

- **sudah / telah** — "çoktan, -di" (bitmiş): `Saya sudah makan`.
- **sedang** — "-makta, -iyor" (sürmekte): `Dia sedang membaca`.
- **akan** — "-ecek" (gelecek): `Kami akan pergi`.
- **belum** — "henüz değil"; **masih** — "hâlâ"; **pernah** — "(hiç) -mişliği".

## 22. İKİLEME (reduplikasyon)

- Çokluk/çeşitlilik: anak → **anak-anak** (çocuklar).
- Karşılıklılık: pukul-memukul (birbirini dövmek).
- Türetilmiş anlam: jalan → **jalan-jalan** (gezinmek).
- Sayı/niceleyiciden sonra çokluk için ikileme yapılmaz.

# PARTİ 5 — BAĞLAÇLAR, EMİR, AKTARIM (Böl. 21-29)

## 23. EMİR KİPİ (Böl. 28)

- Çıplak kök ya da öneksiz meN-fiil: `Makan!`, `Baca buku ini!`.
- Yumuşatıcılar: **-lah** (`Duduklah` = otur lütfen), **tolong/coba/silakan**
  (lütfen/buyurun), **mari/ayo** (hadi).
- **Olumsuz emir**: **jangan** + fiil (`Jangan lupa!`).
- Kibar rica: **tolong/minta** + fiil (`Tolong tutup pintunya`).

## 24. "ÇÜNKÜ" (Böl. 21)

- **karena / sebab** (çünkü), **gara-gara** (yüzünden, olumsuz),
  **oleh karena itu / maka** (bu yüzden).

## 25. "EĞER" (Böl. 22)

- **kalau / jika / bila / apabila** (eğer), **seandainya / andai** (keşke/
  diyelim ki). `Kalau hujan, saya tinggal di rumah` (Yağmur yağarsa evde
  kalırım). "kalau … maka …" (eğer … o zaman …).

## 26. "-DIĞINDA / İKEN" (Böl. 23)

- **ketika / waktu / saat** (-dığında), **sambil** (aynı anda, eşzamanlı eylem:
  `makan sambil menonton`), **sementara / sedangkan** (öte yandan/iken),
  **setelah/sesudah** (-den sonra), **sebelum** (-den önce), **selama**
  (boyunca), **sejak** (-den beri).

## 27. RAĞMEN / "… DEĞİL …" (Böl. 24)

- **meskipun / walaupun / biarpun** (-e rağmen), **tetapi / tapi** (ama),
  **namun** (ancak), **sedangkan** (oysa).
- **bukan … melainkan …** / **tidak … tetapi …** (… değil …).

## 28. BAĞLAÇ VE ODAK EDATLARI

- **dan** (ve), **atau** (veya), **baik … maupun …** (hem … hem …),
  **juga** (de/da), **saja** (sadece), **pun** (bile), **-lah** (pekiştirme),
  **-kah** (soru). "makin … makin …" (… -dikçe … -er) (Böl. 25).

## 29. AKTARILAN SÖZ (Böl. 29)

- Dolaylı aktarım **bahwa** ile: `Dia berkata **bahwa** dia sakit` (Hasta
  olduğunu söyledi).
- "Söylemek" fiilleri: **bilang** (konuşma dili), **berkata / mengatakan**
  (resmî), **menurut** (-e göre).
- Endonezcede **zaman kaydırması yoktur** (İngilizcedeki backshift yok).

## 30. REGISTER VE İLKE

- Standart **EYD** yazımını kullan; ekleri ve meN- nazalleşmesini her zaman
  doğru uygula.
- **TEMEL İLKE**: ana dili konuşanın gerçekten söyleyeceği **doğal, standart
  Endonezce (bahasa baku)** üret — Çince ya da İngilizceden kelimesi kelimesine
  çeviri ASLA. Anlamı sadık tut, ifadeyi Endonezcenin doğal kalıbına dök.
